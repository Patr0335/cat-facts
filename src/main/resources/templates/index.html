Welcome to Cat Facts Generator!

/getSingle: This endpoint calls a service, that fetches a single fact from cat-fact API, maps the result to a CatFact objectand prints the fact to the browser.

/getTen: This end-point calls a service, that fetches ten facts from cat-fact API, maps the result to a CatFact objects, adds to a Arraylist that can only contain CatFacts and prints the result to the browser.

/getTenSortByDate: This endpoint calls a service that fetches ten facts from cat-fact API, maps the result to a CatFact objects, adds to a CatFact Arraylist and prints the result to the browser -sorted in ascending order by Date of creation.
